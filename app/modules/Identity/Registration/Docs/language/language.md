# ユビキタス言語: ユーザー登録コンテキスト

## Registration Request
- **定義**: 新規ユーザーがアカウントを作成するために送信する情報（氏名、メールアドレス、パスワードなど）を含むリクエスト。
- **属性**:
    - Name (氏名)
    - Email (メールアドレス)
    - Password (パスワード)
    - Password Confirmation (パスワード確認)
- **責務**:
    - ユーザー登録に必要な情報を提供する。

## Register Service
- **定義**: ユーザー登録リクエストを処理し、データ検証、新規ユーザーの作成、永続化を担当するアプリケーションサービス。
- **責務**:
    - Registration Requestの情報を検証する。
    - ユーザーを作成し、データベースに保存する。
    - 登録イベントを発行する。

## User Creator
- **定義**: 新しい`User`エンティティを作成し、永続化メカニズム（リポジトリなど）を通じて保存するドメインサービスまたはリポジトリメソッド。
- **責務**:
    - `User`エンティティの整合性を保ちながら作成する。
    - 作成された`User`を永続化する。
